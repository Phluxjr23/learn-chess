# chess notation
this is part 3/3 of learning chess's basics.

here's how it works:

## them squares
take a chessboard (digitally preferably. where would you GET a physical one?) and look at the top of it.
you will see that there are 64 squares. 8 rows, 8 columns.
assuming you know the alphabet which. please tell me you do, label the first row 'a', second row 'b', and so on until you reach 8.

the first row is the one to the left.

then, lable the columns from bottom to top with 1-8.
it should look like this, where `L` is a light square and `D` is a dark square:

```
╭─────────────────╮ 
│ L D L D L D L D │ 8
│ D L D L D L D L │ 7
│ L D L D L D L D │ 6
│ D L D L D L D L │ 5
│ L D L D L D L D │ 4
│ D L D L D L D L │ 3
│ L D L D L D L D │ 2 
│ D L D L D L D L │ 1 
╰─────────────────╯
  A B C D E F G H
```

## the pieces
the pieces are labled as so:
| letter | piece |
| ---------- | ---------- |
| **none** | pawn |
| n | knight |
| k | king |
| q | queen |
| r | rook |
| b | bishop |

## extras
queenside castle: O-O-O
kingside castle: O-O
check: + (at the end)
check*mate*: # (at the end)
capture: x *(eg, moving a knight to capture a pawn on e6 would result in `nxe6`)*
when a pawn captures, it instead uses the row the pawn was on before instead of a letter. eg, axe5
